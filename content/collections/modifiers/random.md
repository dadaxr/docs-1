## Overview
Picks a _single_ random item from an array or collection.

If you are trying to get _multiple_ random items, consider the [shuffle modifier](/modifiers/shuffle).

## Example

### The YAML
```yaml
arr:
  - Soda
  - Pop
  - Coke
```

### The Template
```
{{ arr | random }}
```

### The Output
```
Soda (maybe)
```
